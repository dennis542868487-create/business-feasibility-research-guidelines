# Privacy

This repository and its Skills-only Plugin contain no server, account system, telemetry, analytics, advertising code, or network client. Bundled calculations run locally when the host permits execution.

The host product may process prompts, conversations, uploaded files, tool calls, and outputs under its own plan, workspace, retention, residency, and model-improvement policies. Installing this package does not override those policies.

Use aggregated operating data whenever possible. Remove customer names, contact details, payment information, government identifiers, exact personal addresses, and employee identifiers before analysis. Keep confidential internal evidence separate from public sources and never place raw commercial or personal data in public Evals, issues, examples, or releases.

The Plugin itself does not transmit or retain data outside the files and host session in which it is used.
